Project Overview

CodeTrack enables users to create repositories, manage commits, track version history, and compare code changes through a web interface. The platform is built to demonstrate how modern version control systems function internally, focusing on scalability, reliability, and collaboration workflows.

This project highlights strong full-stack development skills, including API design, database modeling, and frontend-backend integration.


Frontend : React.js, JavaScript, HTML5, CSS3

Backend : Node.js, Express.js

Database : MongoDB
